# Core Project Overview

Apex Trader is an institutional cryptocurrency and futures charting platform inspired by TradingView and institutional trading terminals. It is divided into two main modules: frontend and backend.

## Project Structure
- `frontend/`: Holds the web app client, landing page, and trading dashboard modules. For frontend-specific details, see `mem:frontend/core`.
- `backend/`: Holds Python service layers, caching, API router, and indicator quant logic. For backend-specific details, see `mem:backend/core`.

## Entry Points
- Dev Server: To start the web server, run `python server.py` (serves the app on port 3000). See `mem:suggested_commands` for local execution and command references.
